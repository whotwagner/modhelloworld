# Just another Hello-World-Kernel-Module for Linux

## Compile

```
make
```

## Install

```
insmod modhelloworld.ko
```

## Check the logs

```
dmesg
```

## Uninstall

```
rmmod modhelloworld.ko
```

## Check the logs again

```
dmesg
```
